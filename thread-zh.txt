🧵 AEO 这个东西，我越研究越觉得会是大事。

SEO 之后的下一个大机会，可能就是它。一个 thread 讲清楚👇

---

1/ 先说个场景。

你对手机说："订下周五北京飞上海，东航优先，靠窗，500以下。"

你的 AI Agent 没打开任何 app，直接调东航 API 拿航班数据、比价、下单、选座——全程没有人类访问过任何网站。

这不是 YY。Chrome 146 已经发了 WebMCP 预览版。

---

2/ AEO = Agent Engine Optimization

如果 SEO 是"让搜索引擎找到你"，AEO 就是"让 AI Agent 用上你"。

从"优化给人看"变成"优化给机器调用"。用户可能永远不会看到你的页面——Agent 读完结构化数据，交易完了，走了。

---

3/ 为什么是现在？

Google 和微软联合推 WebMCP 的 W3C 标准化。这不是小团队的 side project，两大浏览器厂商 all in 了。

WebMCP 对 Agent 的意义，差不多像 HTTP 对浏览器。比喻有点大，但暂时想不到更合适的。

---

4/ 几个数字：

• ChatGPT 周活 4 亿，Agent 模式 Q4 环比增长 340%
• Claude Computer Use 发布后企业 API 调用月增 200%+
• Gartner 说 2027 年 30% 电商交易有 Agent 参与（打个折听吧）

增长曲线很像 2005 年的移动互联网——大多数人觉得"还早"。

---

5/ 如果用户不刷网页了，Google Ads 展示给谁？

可能会变成"Agent 竞价调用"——谁的 API 快、数据准、佣金合理，Agent 就选谁。

广告这门生意要变了。不过具体怎么演化，说实话我也不太确定。

---

6/ 我们做了个 AEO 审计工具，七个维度评估网站 Agent 就绪度：

① Schema.org 结构化数据
② llms.txt
③ robots.txt AI 爬虫策略
④ HTML 语义质量
⑤ WebMCP 就绪度
⑥ OpenAPI 文档
⑦ SSR vs CSR（内容可访问性）

---

7/ 拿这个工具测了大厂，结果挺有意思：

• 淘宝 58分（D）— 最高分，靠多年 SEO 底子
• Apple 50分（D）
• Google 29分（F）— 推 WebMCP 的公司自己没做 😂
• Amazon 29分（F）— 封杀了几乎所有 AI 爬虫

全球头部网站平均分不到 42。

---

8/ 这意味着：蓝海。

在 SEO 赛道跟 Amazon 竞争几乎不可能。但在 AEO 上，一个做得好的中小网站完全可以在 Agent 生态里超过巨头。

窗口期不会太长，但现在肯定不算晚。

---

9/ 现在就能做的事（按优先级）：

🔹 加 Schema.org 结构化数据（30分钟）
🔹 写 llms.txt（15分钟）
🔹 调 robots.txt AI 爬虫策略（15分钟）
🔹 确保关键内容不依赖 JS 渲染（2-8小时）
🔹 改善 HTML 语义（1-4小时）

大部分本来就该做——对 SEO、无障碍、性能都有好处。不亏。

---

10/ AEO 到底有多大我也说不准。可能是下一个 SEO 级别的机会，也可能我高估了 Agent 渗透速度。

但"让机器读懂你的服务并调用它"这个需求不会消失。就算 WebMCP 换了名字，方向不会变。

全球头部网站平均 AEO 分不到 42。现在做的成本最低。
